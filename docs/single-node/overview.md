# Single Node

This section covers everything you can do on a single machine with PRIME-RL.

- [**SFT**](sft.md) — supervised fine-tuning
- [**RL**](rl.md) — reinforcement learning (math, agentic, etc.)
- [**LoRA**](lora.md) — parameter-efficient fine-tuning and multi-adapter training
- [**Distillation**](distillation.md) — on-policy distillation with a teacher model
- [**Multimodal**](multimodal.md) — vision-language model training
